**MedCampus** adalah platform simulasi klinik digital berbasis web yang dirancang khusus untuk mendukung pembelajaran *Problem-Based Learning* (PBL) di lingkungan kampus kesehatan. Aplikasi ini mengintegrasikan manajemen rekam medis elektronik (RME), simulasi kasus klinis, dan fitur telemedicine dasar untuk melatih kompetensi digital dan klinis mahasiswa secara simultan.

## 🚀 Fitur Utama

- **📋 Simulasi Kasus PBL Interaktif**: Mahasiswa dapat mengakses studi kasus klinis bertingkat, mulai dari anamnesa hingga diagnosis.
- **📂 Rekam Medis Elektronik (RME) Edukasi**: Simulasi pengelolaan data pasien yang aman, terstruktur, dan sesuai standar privasi data.
- **💻 Dashboard Klinik Digital**: Antarmuka admin untuk memantau alur pasien, jadwal dokter, dan laporan aktivitas mahasiswa.
- **📊 Analisis & Evaluasi**: Sistem penilaian otomatis berdasarkan ketepatan diagnosis dan kelengkapan pengisian data medis.
- **🔒 Keamanan Data**: Implementasi enkripsi dasar dan manajemen akses berbasis peran (Role-Based Access Control) untuk Dosen, Mahasiswa, dan Pasien Simulasi.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun menggunakan stack teknologi modern:

- **Backend**: [Laravel](https://laravel.com/) (PHP Framework)
- **Frontend**: [Blade Templates](https://laravel.com/docs/blade) / [Tailwind CSS](https://tailwindcss.com/) (atau Vue/React jika digunakan)
- **Database**: MySQL / MariaDB
- **Server**: Apache/Nginx
- **Tools**: Composer, NPM, Git
